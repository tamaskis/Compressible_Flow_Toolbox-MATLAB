prandtl_meyer_inverse

Inverse Prandtl-Meyer function.

Syntax

M = prandtl_meyer_inverse(w)
M = prandtl_meyer_inverse(w,gamma)

Input/Output Parameters

Symbol Variable Description Format Units
Input w angle required to accelerate flow from Mach 1 to the local Mach number 1D
double array		 rad
gamma (OPTIONAL) specific heat ratio, defaults to 1.4 1×1
double		 -
Output M local Mach number 1D
double array		 -

Example #1: Local Mach number given the Prandtl-Meyer angle (in air).

for (in air).

M = prandtl_meyer_inverse(0.4604)

    2.0000
